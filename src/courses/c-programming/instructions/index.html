C INSTRUCTIONS

  • C instruction are the commands in the program that instructs that compiler to do certain action.
  • Basically it gives the instruction to the compiler on how to achive the goals of the progrm.
  • For example,when we add two numbers,the C compiler will not understand how to do it. but if we the write same in terms of code / command using arithmatic operater'+'And numbers or variable,it gives meaningful information to the compiler on how to get the sum.

TYPES OF INSTRUCTIONS

  • Ther are basically three types of instruction in C:
    • Type Declaration Instruction – To declare the type of variables used in a C program.
    • Arithmetic Instruction – To perform arithmetic operations between constants and variables.
    • Control Instruction – To control the sequence of execution of various statements in a C program.

TYPE DECLARATION INSTRUCTIONS

  • These instruction inform the compiler about the type of variable used.
  • That means,whenever a variables is used in the programe,we have to specify what types of data it can hold-like integer,float,double,character etc.
  • This makes the compiler to share only those specify types of values in it.
  • For example, a variable intVar1 declared as int will accept and store only integer values.
  • It will never accept float or character values.
  • Declaration of variable is done at the beginning, inside the main function.
  • This function is the starting point of any C program and all the variable used with main function needs to be declare at the beginning of it,before using in the function.
  • Any other function in the program can also have variable and they needs to be declared at the beginning of respective functions.
  • For Example
    • int a;
    • When we declare any variable at that time we are also eligible to initialize this variable.
      • int i=5,j=7;
      • float a=5.5;
    • when we declare and initialize variable together the issue of ordering occurs. sometime order of declartion matters and sometime not.
      • int i=10,j=25;
      • float b=a+3.1,a=1.5;

ARITHMATIC INSTRUCTION

  • An Arithmetic instruction consists of a variable name on the left hand side of equal to (=) and variable names & condition on the right hand sideof equal to(=).
  • The variables and constants appearing on the right hand side of equal to(=) are connected by arithmatic operators like +,-,*,and/.
  • For Example:-
    • Sum=10+20+30;
    • avg=(a+b+c+num)/4;

CONTROL INSTRUCTION

  • The 'Control Instruction' enable us to specify the order in which the various instruction in a program are to be executed by the computer.
  • In other words the control instruction determine the 'flow of control'in a program.
  • There are four types of control instruction in C They are:-
    • sequence Control instruction
    • Selection or decision control Instruction
    • Repetition or Loop Control Instruction
    • Case Control Instruction
